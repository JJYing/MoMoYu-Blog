# AGENTS
这是一个基于 Hugo 的博客项目，使用 GitHub Pages 部署。

## 项目结构
- `content/posts`: 包含前置元数据的 Markdown 文章。
- `layouts`: Hugo 模板（列表/单页/基础）。
- `static`: 静态资源。
- `public`: 构建输出（请勿手动编辑）。

## 常见任务
- 编辑文章：更新 `content/posts/*.md` 中的前置元数据。
- 模板修改：编辑 `layouts/_default/*.html`。

## 风格和安全
- 优先进行最小化、有针对性的编辑。
- 除非明确要求，否则避免触碰 `public/` 目录。

## 代码风格
- 使用单个 Tab 进行缩进，不使用空格缩进。
- Tab 宽度设置为 2 个字符位置。
- 每一级缩进使用一个 Tab 字符。
- 删除行尾多余空格。
- 文件末尾必须有空行。

## 编码与换行（强制）
- 所有文本文件必须使用 `UTF-8 (no BOM)` 编码。
- 所有文件行尾必须使用 `LF (\n)`，禁止 `CRLF (\r\n)`。
- 禁止将任何文件写回为 `UTF-8 with BOM`，禁止引入或保留 BOM。
- 每次编辑后必须确保编码与换行不被破坏；如发现异常，优先修复后再提交。
